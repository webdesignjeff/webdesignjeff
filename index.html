Sample News Page

This is a sample news page created in my classes at CHS Code Camp, designed to highlight features such as dropdown menus (Nav Item 2), scrollbars (the Upcoming News section), and images changing size when hovered over (the pictures of Nick Cage). The code was written in basic CSS, and the images were provided by various filler image generator sites.

Star Wars Armada Scoreboard

This is a scoreboard app created for the tabletop game Star Wars Armada by Fantasy Flight Games, written with SCSS and jQuery. The score buttons open a prompt for how many points scored, which shows in the larger numbers in the middle of the page. The smaller numbers towards the top are used in tournaments and are calculated based on the difference in score.

Warrior Knights Scoreboard

This is a scoreboard created for the game Warrior Knights by Fantasy Flight Games, written with SCSS and jQuery. In this game, you score one point for cities you control from a limited pool of points. I wrote this not just as a scoreboard, but to project the final scores and how many turns the game would last.

Star Wars Rebellion Map Tracker

This is a map tracker created for the game Star Wars Rebellion by Fantasy Flight Games. In this game, the Imperial player searches for the Rebel base. This map, written with SCSS and jQuery, is helps the Imperial player flag locations that they've confirmed are not the Rebel Base. Additionally, there is a function to calculate how many units both players produce.
